Installation Requirements

To install Writing Wrodpress theme, you need:

  • Wordperss 3.5 or higher.
  • Running webserver and working version of WordPress installed.
  • PHP and MySQL Installed on the server.
  • Purchasing Writing theme licence from Themeforest.

How To Install Writing

Now we assume that you already have WordPress installed and working on your server and you have Writing package you purchased from Themeforest, if you didn’t have WordPress Site working yet, we advice you to follow this page which has been provided by WordPress to show you installation process details. if you already has it installed so it’s time to start installing Writing, simply choose one of the following 2 ways to install the theme.

Install Via WordPress Control Panel

  1. Download the package you purchased from Themeforest, and extract the .zip in your computer.
  2. Login to your WordPress control panel and click on the “Appearance” tab then go to “Themes” page.
  3. From the top of the page click on “Add New” button then click on “Upload Theme”.
  4. Choose writing.zip file from your computer and click “Install Now”.
  5. After theme installation complete click “Activate The Theme” and now theme should be ready to work.
Some times you are not able to installing the theme by uploading the .zip file like the previous steps, it’s due to your server limitations which limit the max size upload of the file, so if you have an access to your server configuration files we recommend that you increase max_execution_time – memory_limit – post_max_size – upload_max_filesize if you can’t modify them then you need to go the second way of installation.

Uploading Via FTP

  1. Download the package you purchased from Themeforest, and extract the .zip in your computer.
  2. Use FTP client to access your WordPress installation directory.
  3. Now you need to extract writing.zip file in your computer.
  4. Then upload the entire folder Writing to this path inside WordPress installation ‘wp-content/themes’.
  5. From WordPress control panel go to “Appearance” then “Themes” and you will find Writing theme in the listed themes
  6. Click “Activate” and now theme should be ready to work.

Using Writing Options

Writing is using Wordpress Customizer to make it simple to customize your blog pages, from WordPress control panel go to “Appearance” then “Customize” so you can change your blog layout, upload your own logo and images and change colors

  • Layout: In this section you customize your blog style, set sidebar settings, or select pagination style
  • Social Settings: Here you add set social networks settings, you need to add facebook App ID and Twitter API keys to get the most of social features, you can also add links to your social profile and they will appear in your blog header.
  • Site Title & Tagline: Here you set your site title, and description, you can also add copyright text to your site footer from this section.
  • Colors: Here you can change the main color of your blog.
  • Header Image: Here you can upload your own logo, and you can upload your profile picture.
  • Navigation: Here you can set with menu you like to show in your blog header.
  • Widgets: Here you can add widgets to your blog sidebars and footer area.
  • Static Front Page: Here you can select different page to work as your blog homepage.

Creating Site Pages

Additional to creating default pages for your site, you can add Blog template, this template will create a page which shows latest blog posts with in many styles.

If you want to select another template than the default one, just from “Page Attribute” Box in the right side, under “Template” select box, you can choose between Default Template or Blog Page.

Setting Up Home Page

By default WordPress home page shows latest blog posts, if you want to create a home page you need to create it as any other page, select it’s template and add content to it, but after creating it you need to set it as home page so WordPress understand that this page is the home page of the site, to set up the home page you need to

  1. From WordPress control panel go to “Settings” then “Reading”.
  2. Under “Front page displays” option select “A static page”.
  3. From the “Front page” dropdown below select one of the pages you created and it will work as home page.
  4. Click “Save Changes” and enjoy your site’s home page.

Adding Posts

After creating and managing your pages, now you are ready to add posts to your blog, we will guide you on how to add a post or a page.

Adding Post

To add posts you need to make sure that you installed and activated Post Formats plugin included in the theme, if you didn’t install it please read about “Installing Recommended Plugins” in “How To Install Writing” section, if you already have the plugin activated then you can start now.

  1. Go to Posts then Add New to create new post.
  2. From the right side select the post format (label 1).
  3. Write your post title (label 2).
  4. In the editor type the content of the post (label 3).
  5. Under General Page Options box you can control this post option (label 4).
  6. From right side you need to add this post to one or more categories (label 5).
  7. Then add tags to the post if you want (label 6).
  8. Then upload featured image to represent the post (label 7).
  9. Now click “Publish” to save the post (label 8).

Post Formats

If you want to change the banner of the blog post or the project you can use Post Fortmats, you can select post formats from tabs above the project and post title, and you can choose between

  • Standard: doesn’t show any banner, only text and description.
  • Image: uses post featured image of the post and adds it as a banner before the content.
  • Video: adds embedded video on the top of the post
  • Gallery: adds image slider above the content.
  • Audio: adds an embedded audio to the top of the post.
If you use Standard, Audio, Gallery or Video post format then you need to set Featured Image to the post too so it can be used as the post thumbnail.
