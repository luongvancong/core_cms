# Get started
### Config docker env
- Go to folder `docker`
- Copy `.env.example` to `.env`
- Fill `.env` with your config
### Config app env
- Copy `.env.example` to `.env`
- Fill `.env` with your config
### Create permissions
- To define a new permission, go to `config/permissions.php`
- Run `php artisan permissions:sync` to sync permissions to database
